Sistema de gestion de ventas para supermercados

Este proyecto implementa un sistema de gestion de ventas para una cadena de supermercados. Fue desarrollado como ejercicio de programacion estructurada en C#, utilizando conceptos fundamentales como funciones, vectores, matrices, manejo de menus, validacion de datos y organizacion modular del codigo.

Descripcion del sistema

El programa permite gestionar informacion clave de un supermercado:

-configurar y registrar sucursales

-configurar y registrar productos

-registrar ventas por cada sucursal y producto

-calcular totales de ventas utilizando matrices

-identificar el producto mas vendido

-generar informes en pantalla con formato y colores

Todo el flujo se controla desde un menu principal, y el sistema valida cada entrada para evitar errores. Se emplean arreglos y una matriz bidimensional para almacenar las ventas, fortaleciendo el uso de estructuras basicas de datos.

Objetivos academicos

Este ejercicio permite a los estudiantes:

-aplicar estructuras secuenciales, condicionales y repetitivas

-trabajar con funciones para modularizar el codigo

-utilizar vectores y matrices en un contexto real

-generar reportes a partir de datos almacenados

-manejar la consola con colores, limpiezas de pantalla y titulos

-fortalecer logica de programacion y buenas practicas

Ejecucion

Compila y ejecuta el programa en cualquier proyecto de consola en C#.

El menu guiara todo el proceso de registro y consulta.
